# 常见的 HTTP 状态码
---
- **1xx**：请求正在处理
- **2xx**：请求正常处理完毕
    - **200**：请求成功
    - **201**：通常是 POST 请求成功
- **3xx**：重定向
    - **301**：永久重定向
    - **302**：临时重定向
    - **304**：资源未修改，继续使用缓存资源
- **4xx**：客户端错误
    - **400**：请求语法有误
    - **401**：身份验证未通过
    - **403**：拒绝请求
    - **404**：找不到资源
    - **405**：请求方法不支持
- **5xx**：服务器错误
    - **500**：服务器内部错误
    - **502**：服务器崩溃
