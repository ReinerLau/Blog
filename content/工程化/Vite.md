# Vite

## 定义

解决 webpack 项目越大，启动和更新越慢的问题

## 原理

- 跳过打包

   - 浏览器支持 ESM 规范

   - 将分析依赖关系的工作交给浏览器

- 依赖预构建（首次启动）

   - 模块兼容性：将非 ESM 规范的第三方依赖转换为 ESM 规范

   - 合并：将第三方依赖的多个内部模块合并成单个模块，减少 HTTP 请求

   - 强缓存：利用 HTTP 头 max-age

- esbuild: 使用 GO 编写，比 JS 编写的构建工具快，但打包不够完善，需要 Rollup 支持
